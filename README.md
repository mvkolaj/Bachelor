# Bachelor Thesis: Cross-Genre Music Collaboration Success Prediction

This repository contains the code and final report of our bachelor thesis at the IT University of Copenhagen. The thesis was supervised by Maria Sinziiana Astefanoaei.

## About
This project investigates whether genre distance, measured via hyperbolic embeddings, provides statistically significant predictive value in predicting the streaming success of cross-genre music collaborations.
Research Question: "Does genre distance provide statistically significant predictive value in collaboration success prediction?"
The project combines Spotify chart data (2017-2019), genre collaboration networks, and ENOA hyperbolic embeddings to build prediction models using machine learning techniques.

## Technical Backbone
The hyperbolic genre embeddings implementation is based on the Lorentz model from:
https://github.com/nickboyko/lorentz-genre-embeddings
